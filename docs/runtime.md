# Runtime

Milestone 7 runtime images are generated by `scripts/build_runtime_sw.py`. The script is a deterministic repository-local RV32I/LRSC assembler and image writer used because the local environment provides Apple clang but no complete RISC-V GCC/LLVM objcopy/objdump flow.

The runtime starts all four harts at reset, jumps to text at `0x800`, reads hart ID from `0x10000000`, assigns 256-byte per-hart stacks below `0x10000`, initializes shared state on hart 0, releases secondary harts through `0x308`, filters participation by configured active count, and writes status/completion through uncached apertures. Shared workload data begins at `0x1000`.

The generated instruction set is RV32I plus `lr.w` and `sc.w`; no compressed, multiply/divide, floating-point, privileged, or other AMO instructions are allowed. `sw/runtime/` and `sw/linker/sparrow_cluster.ld` define the intended libc-free ABI for a future ELF build.
